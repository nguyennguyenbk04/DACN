\section{Scope}
The scope of Quizum covers the development of a foundational learning-support platform that processes user-provided materials and generates useful study aids. In this stage, the system focuses on core functionalities such as content upload, transcription, summarization, and basic topic extraction. It also includes essential user-facing features like displaying processed results and providing simple interactive tools. Advanced capabilities—such as mind-map generation, quiz creation, AI chatbot interaction, personalized learning insights, and cloud migration to AWS—are identified for future development and are not fully implemented in this phase. The current scope ensures a stable, functional base that can be expanded with more intelligent learning features in the next stages of the project.
