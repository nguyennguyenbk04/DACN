\section{Non-functional requirements}
\subsection{Performance}
\begin{itemize}
    \item Response time: Process a 10-minute video in under 2 minutes (transcription + summarization + quiz generation) on standard hardware
    \item Scalability: Handle videos up to 2 hours long without crashing.
    \item Accuracy: Achieve at least 85\% for transcription (WER metric), 80\% for summary coherence (ROUGE score), and 90\% for quiz relevance (human-evaluated). 
\end{itemize}

\subsection{Usability}
\begin{itemize}
    \item Ease of use: No technical expertise required; provide tooltips, tutorials, and demo videos. 
    \item Localization: Default to English, with options for other languages in UI elements. 
\end{itemize}
