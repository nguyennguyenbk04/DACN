\section{Frontend}
\subsection{Angular}
Angular is a TypeScript-based web application framework developed by Google, designed for building large, scalable, and structured client-side applications. Although not the primary UI framework in Quizum, Angular contributes to the architectural design principles followed in the frontend, emphasizing modularity, component reusability, and clean state management.

\subsection{React}
React is a popular JavaScript library for building interactive user interfaces through a component-based architecture. In Quizum, React serves as the core technology for building dynamic UI components, handling user interactions, managing application state, and ensuring a responsive and intuitive learning experience.

\subsection{Next.js}
Next.js is a React-based framework that provides server-side rendering (SSR), static site generation (SSG), and optimized routing for high-performance web applications. Quizum uses Next.js as the main frontend framework, enabling fast page loads, SEO-friendly rendering, efficient asset handling, and smooth integration with backend APIs. It also simplifies deployment and scalability thanks to built-in optimizations.
